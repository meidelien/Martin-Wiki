Best Software Tool Special Prize

Regardless of the topic, iGEM projects often create or adapt computational tools to move the project forward. Because they are born out of a direct practical need, these software tools (or new computational methods) can be surprisingly useful for other teams. Without necessarily being big or complex, they can make the crucial difference to a project's success.

To compete for the Best Software Tool prize, please describe your work on this page and also fill out the description on the judging form.

To be eligible, your software has to be documented and made available under an OSI approved open source license.

For software resources, please visit the Resources Hub.
